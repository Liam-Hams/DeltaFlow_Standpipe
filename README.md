# DeltaFlow_Standpipe
Code for the Standpipe device & server

Languages: 
  - Arduino-C (ESP32 board)
  - SQLserver (Database)
  - PHP (Server)

Arduino Library Dependencies:
  - ArduinoHTTPClient 
  - Espressif ESP32 Boards Data + Library
  - TinyGSM
  - ESP32 Software Serial
  - Lolin SIM7600 Library

Product: ESP32 4G compatable boards
